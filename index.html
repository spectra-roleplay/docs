    Welkome op de Spectra docs¶

    Hier kan je allemaal informatie vinden over SPECTRA ROLEPLAY.

    Help de docs verbeteren!¶

    Staan er dingen niet op de docs die jij misschien wel kent? Help ons en voeg ze toe.

    Hoe voeg je dingen toe?

    Regels¶

    1. Zet geen informatie neer over criminele locaties. Het is het leukst als ze dat zelf kunnen uitzoeken.

    How to¶

    1. Ga naar spectra-roleplay/docs
    2. Druk op fork. Je hebt nu een fork gemaakt, deze staat nu tussen je repo lijst.
    3. Druk op je profiel foto en klik op Your repositories. Je komt nu in een lijst met met al je repo's
    4. klik op de repo die "docs" heet. Je bent zit nu in jouw kopie van de repo.
    5. Maak veranderingen of voeg dingen toe. De paginas zitten in de map docs hierin zie je meerdere mappen en bestanden, ieder mapje is een subcategorie en ieder md bestand is een pagina.

    ### Hoe pas je een pagina aan

    1. Zoek de pagina
    2. Klik op de pagina
    3. Klik nu op de aanpas knop(potlood icon) Je kunt nu dingen in de pagina aanpassen.
    4. Maak een commit, zorg dat je een klik duidelijke titel geeft!

    ### Hoe voeg je pagina's toe

    1. Ga naar de subcategorie waar jij een pagina toe wil voegen
    2. Klik op add file/create new file
    3. Geef het bestand een naam, hier moet .md achter staan!
    4. zet nu in het bestand wat jij erin wil hebben.
    5. Maak een commit, zorg dat je een klik duidelijke titel geeft!

    Veranderingen doorsturen¶

    1. Klik op pullrequest
    2. Zorg er voor dat het eerste deel hetzelfde is als in de foto.
    3. Klik op create pull request. Zet hier duidelijk neer waarom je deze veranderingen wilt toevoegen.

    Als je de bovenstaande stappen gedaan hebt, dan zal staff ernaar gaan kijken of het goed is. Dan komt het in de docs!
